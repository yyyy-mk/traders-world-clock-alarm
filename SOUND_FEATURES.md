# 🔊 Ver 0.7 - 다양한 알람음 추가

## ✨ 새로운 기능

**6가지 알람음 선택 가능!**
- 별도의 음악 파일 없이 Web Audio API로 생성
- 브라우저만 있으면 즉시 사용 가능
- 모든 알람에 동일하게 적용
- 설정은 브라우저에 자동 저장

## 🎵 알람음 종류

### 1. 🔔 기본 비프음 (Beep)
```
특징: 짧고 간단한 전자음
길이: 0.5초
용도: 빠른 알림, 기본 설정
음높이: 800Hz 단일 톤
```

### 2. 🎵 차임벨 (Chime)
```
특징: 3음계로 이루어진 부드러운 멜로디
길이: 약 1.2초
용도: 부드러운 알림, 일상적인 알람
음계: C-E-G (도-미-솔)
```

### 3. 🔔 벨 소리 (Bell)
```
특징: 교회 종소리 같은 하모닉 울림
길이: 1.5초
용도: 중요한 알람, 아름다운 소리 선호
기술: 5개의 하모닉 주파수 합성
```

### 4. ⏰ 디지털 알람 (Digital)
```
특징: 전자시계 알람음 (삐-빕 반복)
길이: 약 1.2초 (3회 반복)
용도: 명확한 알림, 깨어있게 하는 용도
패턴: 880Hz-988Hz 교차 3회
```

### 5. 🌅 부드러운 소리 (Gentle)
```
특징: 점점 높아지는 잔잔한 소리
길이: 1초
용도: 아침 알람, 부드러운 깨우기
패턴: 400Hz → 800Hz 상승
```

### 6. 🚨 긴급 알람 (Urgent)
```
특징: 사이렌처럼 주의를 끄는 소리
길이: 약 1.2초 (4회 반복)
용도: 매우 중요한 알람, 긴급 상황
패턴: 600Hz-900Hz 빠른 상승 4회
```

```
- 모든 알람에 동일한 소리 적용
```

## 🔧 기술 구현

### Web Audio API 사용
```javascript
// 별도의 음악 파일 필요 없음!
const ctx = new AudioContext();
const oscillator = ctx.createOscillator();
const gainNode = ctx.createGain();

// 주파수, 파형, 볼륨 조절로 다양한 소리 생성
oscillator.frequency.value = 800;  // 음높이
oscillator.type = 'sine';          // 파형 (sine, square, sawtooth)
gainNode.gain.value = 0.3;         // 볼륨
```

### 각 알람음의 특징

#### 기본 비프음
```javascript
- 주파수: 800Hz
- 파형: sine (사인파)
- 감쇠: 지수 감소 (0.3 → 0.01)
- 파일 크기: 0KB (코드로 생성)
```

#### 차임벨
```javascript
- 3개의 음 순차 재생
- 주파수: 523.25Hz (C), 659.25Hz (E), 783.99Hz (G)
- 간격: 각 음 0.3초씩 지연
- 화음: 장3화음
```

#### 벨 소리
```javascript
- 5개의 하모닉 동시 재생
- 기본 주파수: 440Hz (A)
- 배음: 1, 2, 3, 4.2, 5.4배
- 볼륨: 배음별로 점진적 감소
```

#### 디지털 알람
```javascript
- 2개 주파수 교차: 880Hz, 988Hz
- 파형: square (사각파 - 날카로운 소리)
- 반복: 3회
- 각 음: 0.15초, 간격 0.2초
```

#### 부드러운 소리
```javascript
- 주파수 상승: 400Hz → 800Hz
- 지속 시간: 1초
- 파형: sine (부드러운 사인파)
- 감쇠: 지수 감소
```

#### 긴급 알람
```javascript
- 주파수 급상승: 600Hz → 900Hz
- 파형: sawtooth (톱니파 - 긴급한 느낌)
- 반복: 4회
- 각 사이클: 0.15초 상승 + 0.15초 정지
```

## 🔍 문제 해결

### Q: 소리가 안 나와요
```
A: 다음을 확인하세요:
1. 브라우저 볼륨 확인
2. 시스템 볼륨 확인
3. "알람 엔진 시작" 버튼 클릭했는지 확인
4. F12 → Console에서 에러 확인
```

### Q: 소리가 너무 작아요
```
A: Web Audio API는 시스템 볼륨에 따라 작동합니다.
해결책:
1. 시스템 볼륨 증가
2. 브라우저 볼륨 증가
3. 코드에서 gain 값 조절 (고급)
```

### Q: 소리가 너무 커요
```
A: 
1. 시스템 볼륨 감소
2. 브라우저 볼륨 감소
3. "부드러운 소리" 선택 (볼륨 낮음)
```

### Q: 여러 번 울리나요?
```
A: 현재는 1회만 울립니다.
- 기본 비프음: 0.5초 1회
- 차임벨: 3음 1회
- 디지털: 삐-빕 3회 (내장)
- 긴급: 사이렌 4회 (내장)

추가 반복이 필요하면 별도 요청 주세요!
```
