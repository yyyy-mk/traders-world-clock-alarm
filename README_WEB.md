# 🌐 세계시간 알람 & 타이머 (웹 버전)

브라우저에서 바로 실행되는 세계시간 알람 & 타이머!
썸머타임 자동 적용 + 로컬 스토리지로 알람 저장

## ✨ 주요 특징

### 🌍 완전한 웹 기반
- **설치 불필요**: Python이나 패키지 설치 없이 브라우저에서 바로 실행
- **크로스 플랫폼**: Windows, Mac, Linux 모두 지원
- **모바일 지원**: 스마트폰, 태블릿에서도 사용 가능

### ⏰ 강력한 알람 기능
- **썸머타임 자동 적용**: Luxon.js로 정확한 타임존 변환
- **실시간 미리보기**: 설정 시 즉시 한국시간 확인
- **로컬 스토리지 저장**: 브라우저를 닫아도 알람 유지
- **브라우저 알림**: 백그라운드에서도 알림 수신

### ⏱️ 유연한 타이머
- **다중 타이머**: 여러 타이머 동시 실행
- **실시간 카운트다운**: 초 단위 표시
- **빠른 설정**: 1분~4시간 원클릭

## 🚀 사용 방법

### 1. 파일 열기
```
world_time_alarm_web.html 파일을 더블클릭
```
또는 브라우저로 드래그 앤 드롭

### 2. 알림 권한 허용 (최초 1회)
- 노란색 알림 박스가 나타나면 "알림 권한 허용" 클릭
- 브라우저 팝업에서 "허용" 선택

### 3. 알람 설정
1. 타임존 선택 (예: US/Eastern)
2. 시간 입력 (예: 09:30)
3. 한국시간 미리보기 확인
4. 알람 이름 입력
5. 요일 선택
6. "알람 추가" 버튼 클릭

### 4. 타이머 사용
1. "⏱️ 타이머" 탭 클릭
2. 시간 입력 또는 빠른 설정 버튼 클릭
3. "타이머 시작" 버튼 클릭

## ⚠️ 주의사항

### 알람이 안 울릴 때
1. **브라우저 탭 확인**: 탭이 열려있어야 함
2. **브라우저 종료 금지**: 완전 종료하면 알람 중지
3. **알림 권한**: 브라우저 설정에서 알림 허용 확인
4. **절전 모드**: PC가 절전 모드면 알람 안 울림

### 데이터 보관
- 알람은 **브라우저의 로컬 스토리지**에 저장
- 브라우저 데이터 삭제 시 알람도 삭제됨
- 중요한 알람은 스크린샷 저장 권장

### 썸머타임 정확도
- Luxon.js의 IANA 타임존 DB 사용
- **2026년까지 썸머타임 자동 적용 보장**
- 그 이후는 라이브러리 업데이트 필요

## ❓ FAQ

**Q: 오프라인에서 사용 가능한가요?**
A: Luxon.js를 처음 로드한 이후에는 가능합니다. 완전 오프라인 버전을 원하시면 Luxon.js를 로컬에 다운로드하여 HTML 파일과 같은 폴더에 넣으면 됩니다.

**Q: 알람이 정확한가요?**
A: 네! Luxon.js는 IANA 타임존 데이터베이스를 사용하여 썸머타임까지 정확하게 처리합니다.

**Q: 여러 브라우저에서 알람이 동기화되나요?**
A: 아니요. 각 브라우저의 로컬 스토리지는 독립적입니다. 같은 알람을 여러 곳에서 사용하려면 각각 설정해야 합니다.

**Q: 모바일 알람이 백그라운드에서 작동하나요?**
A: 대부분의 모바일 브라우저는 백그라운드 타이머를 제한합니다. 모바일에서는 브라우저 탭을 활성 상태로 유지해야 합니다.
